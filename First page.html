CRIMINAL MANAGEMENT






  1. The “Crime Management System” is a web based application for online complaining and computerized management of crime records.
  2. Here in this website a person who wishes to file a complaint or report an incident must register before log in and once the admin authenticates the user he or she can login into the website and file a complaint .
  3. This complaint will be received by police and police can send a message regarding status of the complaint to the user who filed the complaint.
  4. Police can use this software to manage different crimes and some of the works which is done in police station manually. Police gets their login password from admin directly.
  5. Some of the modules like unidentified dead bodies, missing persons, and most wanted criminals can be viewed through the website without logging in.
    6. So this website helps police to find out the problems in the society without them actually coming to the police station.
  6. So we are introducing this website for better finding of records without directly searching for that records and to get them in no time
  7. This saves a lot of time for them to searching for that records and this website makes the records centralized
